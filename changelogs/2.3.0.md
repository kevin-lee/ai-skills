### In `install` and `sync`, offer a renaming option for duplicate skills (#90)

Cases:

- A repository may contain multiple skills with the same name.
  - Users may want to install them under different names.
- A user may also reinstall the same skill.
  - In that case, they likely want to install it under a different name.

Now it offers a renaming option for duplicate skills.
